В чате можно кликнуть правой клавишей мышки по нику игрока, чтобы открылось меню.

Чтобы пересоздать локацию кликнете по ее входу с зажатой клавишей Ctrl.

Список команд чата:

  • [Enter] = activate local chat
  • [Shift+Enter] = activate global chat
  • [Ctrl+Enter] = whisper/reply last whisper
  • /help
